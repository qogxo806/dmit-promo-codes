3. In the shopping cart, locate the **"Apply Promo Code"** input field

A few things that trip people up: the code won't apply if your billing cycle doesn't meet the minimum requirement. If you selected monthly and the code needs quarterly, switching the billing cycle in cart will let it work. Also, most codes are one per account — if you've used the same code before on a different plan, it may not apply again.
